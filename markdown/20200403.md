Leo Editor 已從當前的 6.1 版本更新成 6.2.1版

<!-- PELICAN_END_SUMMARY -->

先解除安裝 Leo 6.1 版本
<pre class="brush: python">
pip uninstall leo
</pre>
再執行
<pre class="brush: python">
pip install leo==6.2.1
</pre>
安裝 Leo 6.2.1
